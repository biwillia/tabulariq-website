Easy Guide

Tabular IQ Installation Guide

Tabular IQ is very easy to install. It only takes a few minutes from the time you download to the time you can start working with your data. Below are the steps you can use to install the software:

Download the software

from the Tabular IQ website. Once the download is completed, the setup file will appear on your computer.

Locate the setup file icon

and double-click it to begin the installation process. A installation wizard will appear.

Click next

to begin the installation process.

Please read

The end-user license agreement. If you agree to the terms, add a check to the checkbox on the bottom-left side entitled "I accept the terms in the License Agreement" and then click Next.

Choose which directory

you would like to install into. Normally, application files are stored in the program files directory by default if you are running Windows. If you'd like to choose an alternate location, click the Browse button and select your location preference. When you are finished, click Next.

Ready

You are now ready to actually install the software. Click Install to continue.

The installation

will begin. It should only take a few minutes to complete.

Completed

Once the installation is complete, click Finish to close the installation wizard.
