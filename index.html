Built for Business Analysts and
IT Pros

Tabular IQ™ is an ad hoc data analysis and reporting tool built for people who work with, manipulate and analyze structured data. Tabular IQ is easy to use, has the power you need to work with large files, and helps you get your work done faster:

  • Access data from anywhere (Oracle, SQL Server, CSV, XLS, EBCDIC, HTML, etc.)
  • Freedom from file size limitations (60 billion rows and 18 petabytes per table)
  • Speed, flexibility, and agility (designed for data usability, not data management)
