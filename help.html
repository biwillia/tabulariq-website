How to Get Help

Help Overview

We'd like to make it as easy as possible for you to start working with Tabular IQ™ and applying it to solving your issues related to your data. We have a few online resources that help in this process:

Tabular IQ Support Forums:

This is a great place to start. If you have a question, try searching for an answer, as it may have already been asked before. Or, if you don't find a previous answer, please feel free to post a new question. We try to be as prompt as possible at replying — if someone from the community doesn't answer first. This is your best place to start for help questions.

Video Tutorials and Demonstrations:

This is the archive for all of our videos, whether tutorials, examples or demonstrations. There is a lot of good information here and can give you a lot of pointers in just a few minutes of watching.

Technical Documentation:

This is our online help manual, which can help you learn about all the various features and functions of the software. This help manual also comes included in the Tabular IQ download; but this online manual is always the most up-to-date.

Technical Specifications:

This area provides the system requirements for using Tabular IQ as well as other technical information.

Feedback:

If you would like to report a bug or have some feedback for how we can improve the product, please let us know via the Feedback form; it'll go straight to our developers.

Contact Us:

If the forums or technical documentation don't provide you the solution you are looking for, please contact us by sending an email to support at tabulariq.com. We try to answer all email within one business day.

Don’t hesitate, contact us for better help and services.
