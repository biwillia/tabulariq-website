Viewing Data Sets

         

Accessing and Saving Data > Viewing Data Sets

 

Each data set that is opened or imported almost always initially appears in a Table view, which is the standard datasheet grid.  However, there are two other views of the data, depending on the data type.  All tables also have a Design (or structure) view, that enables you to control the structure of the table.  Additionally, text-delimited and fixed-length data have a Source view, which is used for setting the parameters for parsing the text.
