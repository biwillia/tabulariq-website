Avg()

         

Working with Formulas > Functions Used in Formulas > List Functions > Avg()

 

AVG() calculates the average of the numbers in the specified list.

 

 

Function Format

 

AVG(number1 [, number2, number3, ...]) 

 

 

Return Value

 

AVG() returns a numeric value 

 

 

Examples

 

AVG(3, 1, 2) = 2 

 

AVG(3, 1, 4, 2) = 2.5 

 

AVG(-3, 1, -4, 2) = -1
