Browsing the Web

         

 

Tabular IQ is a data browser.  This means that along with many tools to access, manipulate and analyze data, you can also access the World Wide Web and your intranet with the built-in browser.  Fusing the data handling abilities of a relational database and the access of a browser, Tabular IQ offers new ways to work with data from the web.  However, you can simply browse the web just like you would with other browsers like Microsoft Internet Explorer or Mozilla Firefox.
