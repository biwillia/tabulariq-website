Cos()

         

Working with Formulas > Functions Used in Formulas > Numeric Functions > Cos()

 

COS() returns the cosine of a number, where the input number is in units of radians.

 

 

Function Format

 

COS(number) 

 

 

Return Value

 

COS() returns a numeric value 

 

 

Examples

 

COS(0) = 1 

 

COS(PI()) = -1 

 

COS(PI()/2) = 0 

 

COS(100) = 0.8623
