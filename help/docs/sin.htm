Sin()

         

Working with Formulas > Functions Used in Formulas > Numeric Functions > Sin()

 

SIN() returns the sine of a number, where the input number is in units of radians.

 

 

Function Format

 

SIN(number) 

 

 

Return Value

 

SIN() returns a numeric value 

 

 

Examples

 

SIN(0) = 0 

 

SIN(PI()) = 0 

 

SIN(PI()/2) = 1 

 

SIN(100) = -0.5064
