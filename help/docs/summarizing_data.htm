Summarizing Data

         

Performing Standard Data Operations > Grouping and Summarizing Data > Summarizing Data

 

Often, it is very useful to identify statistics for a data set you are working with.  You can use the Summarize tool to perform this operation.

 

 

Using the above table, we can get the summary information by clicking the "Summarize" item from the Data menu.  We would get the following result:

 

 

The summary table shows each of the field names along with a variety of aggregate information, including minimum and maximum values, the sum and average, minimum and maximum field width, the count of empty fields (if any) and the total record count for the table.

 

In addition to summarizing the entire table, you may also summarize individual fields by highlighting the specific fields and either selecting "Summarize" from the Data menu or by right-clicking on the field header and clicking on the "Summarize" option.
