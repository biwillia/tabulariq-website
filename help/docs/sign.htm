Sign()

         

Working with Formulas > Functions Used in Formulas > Numeric Functions > Sign()

 

SIGN() returns the sign of a number.

 

 

Function Format

 

SIGN(number) 

 

 

Return Value

 

SIGN() returns an integer 

 

 

Examples

 

SIGN(-42) = -1 

 

SIGN(0) = 0 

 

SIGN(33) = 1
