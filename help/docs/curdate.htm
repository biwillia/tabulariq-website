Curdate()

         

Working with Formulas > Functions Used in Formulas > Date/Time Functions > Curdate()

 

CURDATE() returns the current system date.

 

 

Function Format

 

CURDATE() 

 

 

Return Value

 

CURDATE() returns a date 

 

 

Examples

 

CURDATE() = 12/25/2007

 

(if we were using this function on December 25, 2007)
