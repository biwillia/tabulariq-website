Keyboard Shortcut Reference

         

 

Below are a list of keyboard shortcuts or "hotkeys" that you may use to expedite your data analysis.

 

 

General Keyboard Shortcuts

 

CTRL+O

Open

CTRL+W

Close

CTRL+S

Save

CTRL+D

Bookmark This Location

CTRL+R

Refresh

CTRL+P

Print

CTRL+Z        

Undo

CTRL+Y

Redo

CTRL+X

Cut

CTRL+C

Copy

CTRL+V

Paste

CTRL+A

Select All

CTRL+F

Find

CTRL+H

Replace

CTRL+G

Go To

CTRL+TAB

Next (TAB)

CTRL+SHIFT+TAB

Previous (TAB)

CTRL++

Increase (Zoom)

CTRL+-        

Decrease (Zoom)

CTRL+0        

Normal (Zoom)

CTRL+B

Project Panel

CTRL+K

Console Panel

F11

Full Screen

CTRL+J

Filter

CTRL+M

Remove Sort/Filter

CTRL+E

Copy Records

ALT+ENTER

Run Script/Query

CTRL+BREAK        

Cancel Job

F1

Help Contents

F3

Find Next

SHIFT+F3

Find Previous

CTRL+SHIFT+G

Alternate for Find Previous

F5

Reload

CTRL+ENTER

Complete a URL and Return (When Editing a URL)

CTRL+ENTER

Performs Command for Open Dialog (e.g., Filter, Update)

CTRL+PAGEDOWN

Next (TAB)

CTRL+PAGEUP        

Previous (TAB)

CTRL+U

Toggles Between the Document Views (i.e., Between Source, Structure, and Table View)

ALT+LEFTARROW

Go Forward

ALT+RIGHTARROW

Go Backward

SPACEBAR

Toggle Boolean Checkmark On/Off (When Cursor is on Boolean Field)

 

 

Accessibility Keyboard Shortcuts

 

CTRL+L

Sets Focus to Location Bar

CTRL+/

Sets Focus to Quick Filter/Find

CTRL+I

Sets Focus to the Active Document

CTRL+SHIFT+I

Alternate for Setting Focus to Active Document

CTRL+;        

Shows Console Panel and Sets Focus to Panel

CTRL+:

Alternate for Showing Console Panel and Setting Focus to the Panel

CTRL+SHIFT+K

Shows Console Panel and sets focus to Panel

CTRL+SHIFT+B

Shows Project Panel and sets focus to Panel

CTRL+SPACEBAR

Selects a Column in a Table

SHIFT+SPACEBAR

Selects a Row in a Table
