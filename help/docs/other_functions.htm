Other Functions

         

Working with Formulas > Functions Used in Formulas > Other Functions

 

Other functions are useful tools that don't have a classic category (e.g., creating an IF statement and finding record numbers/identifiers).  Below are the other functions supported in the software:

 

 

Function Name

Type

Description

Empty()

Other

Determines if a field has a null or empty value

Iif()

Other

Returns one of two values based on a Boolean statement

Recid()

Other

Returns a string that uniquely identifies a record

Recno()

Other

Returns a number indicating the absolute position of a record in a data set
