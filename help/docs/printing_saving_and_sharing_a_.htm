Printing, Saving and Sharing a Report

         

Designing and Printing Reports > Printing, Saving and Sharing a Report

 

Once you have finished designing your report, you have the option to save it, print it and share it with others.

 

 

Saving a Report

 

To save a report to your project, select Save from the File menu or the Save icon from the toolbar.  The saved report is associated with the Source table, but acts as a template.  So, if you have a different source table with the same structure, you can apply the saved report to that table.  Also, if your table is refreshed with new data, the report will reflect the new data.

 

 

Printing a Report

 

To print a report, simply select Print from the File menu.  You may also adjust page settings by selecting Page Settings from the File menu.

 

 

Sharing a Report

 

In addition to saving and printing reports, you can also save them to different file formats for sharing with others.  There are two types of files that can be shared:

 

•Portable Document Format (PDF):  PDF is a standard format for sharing documents and uses the ubiquitous Adobe Reader to open and display the PDF.  To create a PDF of your report, select Save As External from the File menu, choose PDF Files as your file type, enter a name for your file and click OK.  The report will be saved as a PDF to your system.

 

•Package Files (KPG):  A Package File lets you send the report template to another person who is also using Tabular IQ.  To save as a KPG, select Save As External from the File menu, choose Package Files as your file type, enter a name for your file and click OK.  The report will be saved as a KPG to your system.  Please note that saving a report in this way only saves the report template and does not save the associated data set.  To save the associated table and the saved report template as a KPG file, see Exporting Data Sets for further information.
