Help Menu

         

Menu Reference > Help Menu

 

Below are explanations and links related to the primary items in the Help Menu:

 

 

Help Contents

 

Opens the help manual supplied with the software.

 

Contact Us

 

Opens the a contact web page, where you can ask support questions or let us know if you have any ideas for improving the software.

 

Activate...

 

Activate your software license.

 

Check for Updates...

 

Checks the download server to see if a new update or upgrade to the software is available.

 

About...

 

Opens the about dialog, which provides further information about the software.
