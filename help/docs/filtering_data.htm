Filtering Data

         

Performing Standard Data Operations > Filtering Data

 

Filtering is a foundational tool of data analysis.  Filtering gives you the ability to create a subset of your table based on certain criteria or formulas.  It is extremely useful for narrowing your data set to only the most appropriate records.

 

For example, let's say we had a table of contact information from across the United States.  Our company wants to send letters to everyone who lives in California, but not to anyone else.  One way to determine the appropriate contacts is by filtering on the state "CA".  This would reduce the number of records we are looking at to only the appropriate records.  An additional nice feature of filtering is that, unlike delete operations, the filter can quickly be removed so the full table is restored.
