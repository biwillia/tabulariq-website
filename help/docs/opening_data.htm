Opening Data

         

Accessing and Saving Data > Opening Data

 

Tabular IQ gives you an ability to open your data files locally (data on your PC) for a wide variety of file types, including CSV, TSV, DBF, and Microsoft Excel (XLS, XLSX) and Access files (MDB, ACCDB).  You can simply open these files in Tabular IQ and begin using them for data analysis.

 

In addition to local data files, Tabular IQ enables you to open and use data files that are located on the World Wide Web or intranet via URLs.  Tabular IQ lets you open CSV, RSS and HTML Tables directly from the web.

 

Tabular IQ also gives you the ability to connect to database systems; see the Creating Connections to Data Sources section for further information.
