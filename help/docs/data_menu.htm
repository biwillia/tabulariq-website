Data Menu

         

Menu Reference > Data Menu

 

Below are explanations and links related to the items in the Data Menu:

 

 

Sort...

 

Opens the sort dialog, which enables you to sort fields in a table.

 

Filter...

 

Opens the filter dialog, which enables you to filter fields in a table.

 

Remove Sort/Filter

 

Removes an existing sort and/or filter from your table.

 

Edit Indexes...

 

Enables saving of indexes (indexes are used in sorts) for future use instead of re-running these operations.

 

Columns

 

Expands into a menu that includes options for inserting and hiding columns and column breaks as well as inserting and editing calculated fields.

 

Groups

 

Expands into a menu that includes options for grouping records as well as inserting and removing group breaks.

 

Mark Records...

 

Opens the marks dialog and panel, which enables you to mark/highlight records in your data based on a formula.

 

Copy Records...

 

Opens the copy dialog, which enables you to copy records from one table to a new table.

 

Append Records...

 

Opens the append dialog, which enables you to append data to your table from a second table.

 

Delete Records...

 

Opens the delete dialog, which enables you to delete records in your table based upon a formula.

 

Update Records...

 

Opens the update dialog, which enables you to update/replace records and values in your table.

 

Summarize

 

Performs a summarizing operation on your data set (or selected fields), which provides aggregate statistics for each field such as minimum, maximum, average sum as well as total and empty counts.
