Using Save As

         

Accessing and Saving Data > Saving Data and Bookmarks > Using Save As

 

In addition to saving a table to a project, you can also use the Save As functionality to make copies of tables.  For example, here are three instances of when you may use Save As:

 

•Save a copy of an existing table.
•Save a copy of an existing table, which has been manipulated with data operations, such as filters.
•Save a copy of a table to a database system.

 

To use Save As, simply open up the table you wish to copy and then select Save As from the File menu.  A Save As dialog will open.  Select the folder you wish to save to (if any) and then enter the name of your new table in the box.  Click the Save button to make a copy of your table.
