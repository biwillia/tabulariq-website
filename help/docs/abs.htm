Abs()

         

Working with Formulas > Functions Used in Formulas > Numeric Functions > Abs()

 

ABS() returns the absolute value of a number.

 

 

Function Format

 

ABS(number) 

 

 

Return Value

 

ABS() returns a numeric value 

 

 

Examples

 

ABS(-42) = 42 

 

ABS(0) = 0 

 

ABS(33) = 33
