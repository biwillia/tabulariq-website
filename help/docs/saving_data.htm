Saving Data and Bookmarks

         

Accessing and Saving Data > Saving Data and Bookmarks

 

Once you have opened your data tables, you may want to save them to your project or transform them into different formats.  Tabular IQ offers you the tools to perform these tasks.  For instance, you can Save an  external data file to a project.  Or you can use Save As to perform such tasks as copying a table to an external database system or saving a copy of a data table that you've filtered.  Tabular IQ also provides the tool Save As External to quickly save the tables you are working on to a variety of external formats.
