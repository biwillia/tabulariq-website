Editing Values in a Cell

         

Performing Standard Data Operations > Editing and Replacing Data > Editing Values in a Cell

 

There are times that you may be working with a data set and want to make manual edits, often to clean up "dirty" data.  Tabular IQ let's you edit the data in cells, just like you might do in a spreadsheet.

 

To edit data in a cell, simply double-click on the blank cell to get a cursor.  Then type in your new text:

 

 

NOTE:  In order to edit a cell, you must first go to the Edit menu and make sure that "Protect Data" is turned off (unchecked).
