Opening Data from the Web

         

Accessing and Saving Data > Opening Data > Opening Data from the Web

 

Tabular IQ is a data browser, which means that you can access data directly from the web or intranet using a URL, as simply as you can access it locally.  In particular, Tabular IQ lets you work with the following data types directly from the web:

 

•Text Files (CSV)
•RSS Feeds
•HTML Tables

 

In addition, there may be other data files you encounter on the web that you can first save to your PC and before opening it as a local data file.

 

 

Opening Text-delimited Files from a URL

 

When you encounter a data file on the web it is often in a CSV format for downloading to a spreadsheet.  Tabular IQ lets you open these files directly, by simply clicking on the URL.  To open a CSV file embedded in a URL, do the following:

 

1. Browse to the web page that has a link to the data set you wish to view or simply copy the link into the Tabular IQ browser.

 

2.  A table will open in Tabular IQ, which can be used immediately for analysis.

 

There may be occasions when Tabular IQ will not recognize the delimiter or the proper structure of the fields in the table.  To adjust this, use the Source View functionality to define the widths of each field and then use the Design View to define the structure for each field.

 

Additionally, some file types will not open automatically into a table.  To view these files, first save the file to your desktop and then open it up as you would with standard local data file.

 

Opening RSS Feeds from a URL

 

RSS Feeds are used on the web for quickly syndicating blog and news items.  Normally, these files are parsed by a feed reader for browsing and reading the text.  However, Tabular IQ enables you to open and use theses files for your data analysis, just like any other table.  To open an RSS feed, do the following:

 

1. Browse to the web page that has a link to the RSS feed you wish to view or simply copy the RSS URL into the Tabular IQ browser.

 

2.  A table will open in Tabular IQ, which can be used immediately for analysis.

 

There may be occasions when Tabular IQ will not recognize the delimiter or the proper structure of the fields in the table.  To adjust this, use the Source View functionality to define the widths of each field and then use the Design View to define the structure for each field.

 

 

Opening HTML Tables from a Web Page

 

Often web pages use HTML markup to create data tables for their readers to use.  It is usually quite difficult to get access to this type of data, since it is embedded into the web page.  However,  Tabular IQ lets you easily open and use these files.  To open an HTML table, do the following:

 

1. Browse to the web page that has the HTML table you wish to view.

 

2.  Right-click on the HTML table and click Import.  A table will open in Tabular IQ, which can be used immediately for analysis.

 

There may be occasions when Tabular IQ will not recognize the structure of the fields in the table.  To adjust this, use the Source View functionality to define the widths of each field and then use the Design View to define the structure for each field.
