Tutorials

These videos provide instructions how to use specific features:


Getting Started with Tabular IQ™

Summary: This video offers a short guide to help you get around Tabular IQ's controls and tools.  It covers a wide range of areas including projects, tabs, panels, menus and views.
Topics: projects, web browsing, views, editing data, adding records


Opening and Saving Data Sets

Summary: Learn how to work with local data files (e.g., CSV, DBF), connect to external databases (e.g., Oracle, SQL Server), and use Web data (e.g., HTML tables, RSS).
Topics: local data, external databases, converting, exporting


Working with Calculated Fields

Summary: Learn how to use calculated fields for effective analysis.  In part I, learn the basics; in part II, learn to apply functions for more advanced data cleansing.
Topics: calculated fields, functions


Sorting Data

Summary: Learn how to sort fields in a certain order (such as alphabetically or numerically), different sorting options, as well as how to save custom sort orders (indexes).
Topics: sorting, saving indexes


Filtering Data

Summary: Filtering enables you to reduce a data set to a smaller number of records based on some criteria.  Learn how to filter your data with Tabular IQ's quick filter and cell-criteria filter as well as with custom formulas.
Topics: quick filter, filter builder, functions


Creating Reports

Summary: Learn how to quickly create reports, similar to how you would with a spreadsheet.  Also learn how to format, add groups and sort orders, switch views, print and save as a PDF file.
Topics: report settings, sorts and groups, report views, formatting, printing

Examples and Demonstrations

These videos show different ways to use Tabular IQ for various applications:


Explore a New Data Table

Summary: When you have some new data, it pays to do a few tests up front to make sure the set is complete and accurate.  This video explores identifying issues using summaries and date range checks.  More »
Topics: summarize, quick find, delete, calculated fields


Using Multiple Databases from a Single Interface

Summary: Different corporate systems run on different databases (e.g., Oracle, MySQL).  Tabular IQ enables you to access and work with your data from multiple systems at once from a single interface.  More »
Topics: create connections, filter, marks, reports, copy, sql statements


Cleaning Up Dirty Data

Summary: Data with errors and inconsistent formatting is more common than one would hope.  This video shows how Tabular IQ makes it easy to clean up "dirty data" -- in this case phone number formatting.  More »
Topics: calculated fields, functions, grouping


Working With Large Amounts of Data

Summary: Unlike most desktop data tools, Tabular IQ does not suffer from row limitations or 2 gigabyte file restrictions.  This video shows multiple processes using a 7 million record file.  More »
 Topics: filter, group breaks, calculated fields, marks


Creating Reports Quickly

Summary: Tabular IQ makes it really easy to create quick, ad hoc reports.  This video shows how  it's spreadsheet like feel makes it simple to create sophisticated reports in minutes.  More »
 Topics: quick report, print preview, template view, groups and sorts


Opening and Using Text-delimited Files (e.g., CSV)

Summary: Text files, such as CSV, TSV, fixed-length, and EBCDIC files, are commonly used to share data.  Tabular IQ provides and extremely intuitive tool to open and work with text files.  More »
 Topics: open files, table design view, filter, sort, marks, convert data


Converting and Transforming EBCDIC files to ASCII

Summary: EBCDIC is a mainframe format that often needs to be converted before being used in other tools.  Tabular IQ makes it really easy to convert EBCDIC tables to ASCII.  More »
 Topics: open files, table design view, columns, filter, marks, save

Extensions

Benford's Law Extension

Summary: Benford's law describes a mathematical phenomenon involving the first digits of numbers and is often used to help identify fraud. This video walks through some examples; please see related post here.

Apache Log Import Extension

Summary: This extension offers an easy way to import Apache web logs and handle all the parsing and data formatting for you.  This video shows how we identified a problem in our logs; please see related post here.

Lokad Forecasting Extension
