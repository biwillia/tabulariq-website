Putting Together a "Hello, World" Extension

To make a "Hello, world" extension, the first step is to create a script in Tabular IQ that displays the text "Hello, world". Fortunately, this involves just a few steps and a single line of code:

  • . Click New in the File menu, and select Script from the set of options
  • . main.js Add the following line to the new script: 
    •  alert("Hello, world");
  • . main.js Select Save from the File menu and then name the saved file "Hello_world"

At this point, you can press to run this script; it should display an message box that says "Hello, world". If this works properly, select Save As External from the File menu and save this file as "main.js" on your local drive.

The next step is to create an "info.xml" file for the extension. To do this, open Notepad or another text editor and cut and paste the following text into it: 

<?xml version="1.0">
<extension_info>
     <guid>{ad912db5-97f9-4fba-a983-3aa0bffa8faa}</guid>
     <name>Hello World</name>
     <author>Yours Truly</author>
     <bitmap></bitmap>
     <startup>main.js</startup>
     <copyright>Me</copyright>
     <description>Display a Hello World alert message.</description>
     <major_version>1</major_version>
     <minor_version>0</minor_version>
     <subminor_version>0</subminor_version>
</extension_info>

Next, save this file as "info.xml" in the same area where you put "main.js" (NOTE: if you use Notepad, be sure to change the "Save as type" option to "All Files", so that it saves as an XML file).

Finally, you just need to package everything together as a .kxt file:

  • . Create a .zip file on your local drive called "Hello_world.zip"
  • . Add "main.js" and "info.xml" to "Hello_world.zip"
  • . Change the name of "Hello_world.zip" to "Hello_world.kxt"

The "Hello, world" extension is now complete. To add this extension to Tabular IQ, select Extensions from the Tools menu, click Add Extensions and finally select the "Hello_world.kxt" file.

After adding "Hello_world.kxt" to the Extensions area in Tabular IQ, you can run it immediately by clicking the Start Now button. Additionally, after you close Tabular IQ, the next time you open it, this extension will run automatically. Of course, this gets old quickly, and so you can remove the extension from Tabular IQ by selecting Extensions from the Tools menu and clicking the Uninstall button for "Hello World".
